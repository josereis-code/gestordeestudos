# Kanban de Estudos 🗂️

Ferramenta visual minimalista para gestão 100% manual de estudos. Sem distrações, sem algoritmos complexos — você tem o controle total do seu progresso.

## 🚀 Como Usar

1. **Abra** `index.html` no seu navegador.
2. **Adicione** suas disciplinas e os tópicos que deseja estudar.
3. **Organize** seu progresso arrastando os cards entre as colunas do Kanban:
   - **Backlog**: Conteúdo para estudar em breve.
   - **Estudando**: O que você está focando no momento.
   - **Para Revisar**: Conteúdos que já viu, mas precisam de reforço.
   - **Dominado**: Matéria vencida e consolidada!

## 📁 Estrutura de Arquivos

```
KanbanDeEstudos/
├── index.html              # Entrada da aplicação (SPA)
├── css/
│   └── styles.css          # Design system minimalista e responsivo
└── js/
    ├── config.js           # Configuração das colunas e rotas
    ├── db.js               # Persistência local (localStorage)
    └── app.js              # Lógica da interface e Kanban (Drag & Drop)
```

## 🧠 Filosofia do Projeto

- **Simplicidade**: Interface limpa e direta ao ponto.
- **Autonomia**: Você decide o que move e quando move.
- **Manual**: Nada de IA ou automações — o ritmo de estudo é 100% ditado por você.
- **Visual**: Veja todo o seu edital em uma única tela e identifique gargalos rapidamente.

## 🛠️ Tech Stack

- **HTML5 Semântico**
- **Vanilla JavaScript** (ES6+) — Sem frameworks pesados.
- **CSS3 Moderno** (Custom Properties & Flexbox/Grid).
- **localStorage** — Seus dados ficam salvos 100% no seu navegador.
